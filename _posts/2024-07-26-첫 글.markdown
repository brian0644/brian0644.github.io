# **ML 1주차 정리**
## 1. EDA와 시각화
EDA: 가공하지 않은 원천의 데이터를 탐색하고 분석하는 것 (by 기술통계, 시각화)  
-> 극단적 해석, 지나친 추론, 자의적 해석은 되도록 지양한다.  
* 데이터의 형태와 척도가 분석에 알맞는지 확인 (sanity checking)  
* 데이터 평균, 분산, 분포, 패턴 확인 -> 특성파악  
* 결측값, 이상치 파악 및 보완  
* 변수간 관계성 확인  
* 분석 목적과 방향성 점검 및 보정
